(P\+E\+CL memcache $>$= 0.\+2.\+0) \mbox{\hyperlink{class_memcache_pool_ae5ee9b5ace0ff476c62f2812fa290fb9}{Memcache\+::connect}} — Open memcached server connection \mbox{\hyperlink{patch2_8txt_a3f9992ab8fa942ed02faa63682c3a84c}{string \$host }} Point to the host where memcached is listening for connections. This parameter may also specify other transports like unix\+:///path/to/memcached.sock to use U\+N\+IX domain sockets, in this case port must also be set to 0. 


\begin{DoxyParams}[1]{Parameters}
int & {\em \$port} & \mbox{[}optional\mbox{]} \\
\hline
\end{DoxyParams}
Point to the port where memcached is listening for connections. Set this parameter to 0 when using U\+N\+IX domain sockets. Note\+: port defaults to memcache.\+default\+\_\+port if not specified. For this reason it is wise to specify the port explicitly in this method call. 


\begin{DoxyParams}[1]{Parameters}

\hline
\end{DoxyParams}
Value in seconds which will be used for connecting to the daemon.
